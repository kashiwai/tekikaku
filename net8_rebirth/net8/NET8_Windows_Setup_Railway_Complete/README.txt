================================================================
NET8 Windows PCセットアップパッケージ (Railway本番環境版)
================================================================

作成日: 2025-11-05
環境: Railway本番環境 + GCP Cloud SQL
対象: Windows 10/11 PC（ゲーム機）

================================================================
📋 このパッケージについて
================================================================

このフォルダには、NET8カメラシステムをWindows PCにセットアップ
するために必要な全てのファイルが含まれています。

ngrokは使用せず、Railway本番環境に直接接続します。

================================================================
📦 含まれているファイル
================================================================

【セットアップ用実行ファイル】
✅ setupapp.exe (33MB)
   → slotserver.exe等をC:\serverset\にインストール

✅ GoogleChromeStandaloneEnterprise64.msi (59MB)
   → Google Chrome ブラウザ

【バッチファイル - 順番に実行】
✅ 1_Office_AutoRun.bat
   → 初期セットアップ（PC名変更、電源設定など）
   → 実行後に再起動が必要

✅ 2_Site_AutoRun_railway.bat ★Railway環境用★
   → アプリケーションインストール
   → Railway環境に設定済み（手動変更不要）

✅ 3_Last_AutoRun.bat
   → 自動化設定（自動ログイン、タスクスケジューラ）

【補助ファイル】
- Net8AppInstall.bat
- ChromeLocalInstall.bat
- PowerControl.bat
- PowerShellスクリプト群/

【ドキュメント】
📖 セットアップ手順書.md
   → 完全な手順が記載されています（必読！）

📖 README_RAILWAY.md
   → Railway環境用の補足説明

================================================================
🚀 セットアップ手順（簡易版）
================================================================

【事前準備】
1. このフォルダ全体をWindows PCにコピー
2. PowerShellを管理者として実行

【ステップ1】初期セットアップ + 再起動
→ 1_Office_AutoRun.bat を管理者として実行
   入力: Camera Terminal Number = 1
   入力: Host Number = 1
→ PCを再起動

【ステップ2】アプリケーションインストール
→ 2_Site_AutoRun_railway.bat を管理者として実行
   入力: Host Number = 1

【ステップ3】自動化設定
→ 3_Last_AutoRun.bat を管理者として実行
   入力: Username = pcuser
   入力: Password = pcpass
   入力: Shutdown time = 09:00:00

【ステップ4】Railway設定ファイル作成
→ セットアップ手順書.md の「ステップ4」参照
   PowerShellコマンドをコピー&ペースト

【ステップ5】接続テスト
→ セットアップ手順書.md の「ステップ5」参照

【ステップ6】最終再起動
→ 自動起動テストのため再起動

================================================================
⚠️ 重要な注意事項
================================================================

❌ ngrokは使用しません
✅ Railway本番環境に直接接続します

【接続先】
- Webサーバー: https://mgg-webservice-production.up.railway.app/
- データベース: GCP Cloud SQL (136.116.70.86:3306)

【必ず管理者権限で実行してください】
- すべてのバッチファイルを右クリック
- 「管理者として実行」を選択

【インターネット接続が必要です】
- Railway環境への接続確認
- setupapp.exeのダウンロード（初回のみ）

✅ セットアップ完了の確認
================================================================

以下が全て✅になれば完了です：

□ C:\serverset\slotserver.exe が存在する
□ C:\serverset\slotserver.ini が存在する
□ slotserver.exeが起動している
□ chrome.exeが起動している
□ カメラ映像が配信されている

視聴確認URL:
https://mgg-webservice-production.up.railway.app/play_v2/?NO=1

📦 SDK統合方法（3行で完了）

実際のクライアント企業がSDKを統合する場合、以下の3行をHTMLに追加するだけです：

<!-- 1. SDKを読み込む -->
<script src="https://mgg-webservice-production.up.railway.app/sdk/net8-sdk-beta.js"></script>

<!-- 2. コンテナを用意 -->
<div id="game-container"></div>

<!-- 3. SDKを初期化してゲーム起動 -->
<script>
const net8 = new NET8SDK('pk_test_dummy_partner_2025');
net8.launch('#game-container', { modelId: 'HOKUTO4GO' });
</script>

---

## ステップバイステップ手順

### Step 1: Railway Tokensページにアクセス

**直接リンク**: https://railway.com/account/tokens

または：

1. https://railway.app/ にアクセス
2. 右上のアカウントアイコンをクリック
3. "Account Settings" を選択
4. 左メニューから "Tokens" を選択

---

### Step 2: 新しいトークンを作成

1. **"Create New Token" ボタンをクリック**

2. **トークン名を入力**
   ```
   例: net8-deployment
   ```

3. **"Create" をクリック**

4. **生成されたトークンをコピー**

   ⚠️ **重要**: トークンは一度しか表示されません！

   **正しいトークンの形式例**:
   ```
   rw_Fe26.2**60120c18bf365c1857d9c1297a52944727878ab9602e34a65ec0a5bff6fc462f*...
   ```

   - 先頭が `rw_` で始まる
   - 非常に長い文字列（200文字以上）
   - 暗号化されたデータを含む

---

### Step 3: トークンを提供

生成されたトークン全体をコピーして、こちらに送信してください。

**形式**:
```
rw_Fe26.2**...（全文）
```

## 🚀 トークン取得後の自動デプロイ

正しいAPIトークンを取得したら、以下のコマンドで全自動デプロイを実行します：

```bash
./railway-deploy.sh <APIトークン>
```

---

## 📋 別の方法: Railway CLI認証

APIトークンを手動で入力したくない場合：

```bash
railway login
```

このコマンドを実行すると、ブラウザが開いて自動的に認証されます。

## 🆘 よくある質問

### Q: トークンが見つからない

**A**: https://railway.com/account/tokens にアクセスして新規作成してください

### Q: トークンを保存し忘れた

**A**: 再度新しいトークンを作成してください（古いトークンは削除可能）

### Q: チームトークンと個人トークンの違いは？

**A**:
- **個人トークン**: 個人のすべてのリソースにアクセス可能（推奨）
- **チームトークン**: チーム内のリソースのみアクセス可能

NET8プロジェクトには**個人トークン**を使用してください。

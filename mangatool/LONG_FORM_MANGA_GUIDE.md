# 長編漫画制作ガイド

mangatool で最大 250 ページまでの長編漫画を制作できます。

---

## 📖 **ページ数別の選択肢**

### **短編（16ページ以下）**
- 1, 2, 4, 6, 8, 12, 16 ページ
- 生成時間: 1ページあたり 1～2 分
- 用途: 広告紹介、プロモーション、短いストーリー

### **中編（20～50ページ）**
- 20, 32, 50 ページ
- 生成時間: 1ページあたり 3～5 分
- 用途: 短編作品、テスト販売

### **長編（75～250ページ）** 🚀
- 75, 100, 150, 200, 250 ページ
- 生成時間: 1ページあたり 5～10 分
- 用途: フル作品、書籍化、本格的な連載

## 🚀 **長編制作の流れ**

### **Step 1: 制作方法を選択**

- **URL**: ホームページ・ランディングページから自動解析
- **画像**: キャラクター画像からストーリー生成
- **テキスト**: あらすじから本格制作（推奨）

### **Step 2: ページ数を設定**

```
ドロップダウンメニュー → 「長編（最大250ページ）」から選択
```

### **Step 3: 詳細設定**

✅ **言語**: 日本語 / English / 中文 / 한국어  
✅ **スタイル**: 漫画 / Webtoon / コミック / アニメ  
✅ **表紙**: 自動生成（オプション）  

### **Step 4: 生成開始**

```
「生成を開始」をクリック
  ↓
バックグラウンドで生成
  ↓
完了したらダッシュボードに表示
```

## 💾 **長編制作のコツ**

### **1. ストーリー作成時は詳細に**

テキスト制作の場合、あらすじはなるべく詳しく書くと、AI がより良いストーリーを生成できます。

**良い例:**
```
田中太郎は新入社員。最初の3か月は失敗が続き、同僚には馬鹿にされていた。
しかし先輩の鈴木さんに励まされ、営業スキルを学ぶ。
3か月後、大型案件を自分一人で成功させ、配属部門の売上を30%増加させた。
最終的には営業部のエースとなり、チームのリーダーに抜擢される。
```

**悪い例:**
```
新入社員が頑張って成長する話
```

### **2. キャラクター設定の継承**

長編では、以下が自動で継承されます：
- 主人公のキャラクター設定
- 色使い・画風
- トーン・雰囲気

### **3. 表紙の活用**

長編制作時は表紙の自動生成を有効にすることをお勧めします：
- 📖 「表紙（カバー）」をオン
- 🎨 「カラー」を選択（本格的に見える）
- ➕ 表紙 1 ページ + 本編ページ

## 🔄 **続編・シリーズ化**

完成した漫画から続編を作成できます：

1. 漫画詳細ページで「続きを作る」をクリック
2. ページ数を設定（新作は独立したページ数）
3. 前作のキャラクター・設定を自動継承

**例:**
```
第 1 巻: 100 ページ（発売）
  ↓
第 2 巻: 150 ページ（「続きを作る」で制作）
  ↓
第 3 巻: 150 ページ（シリーズ化）
```

## ⚠️ **長編制作時の注意**

### **1. 生成時間が長い**

- 250 ページの漫画は最大 8 時間かかります
- 夜間に生成開始することをお勧めします
- バックグラウンドで動作するので、ブラウザを閉じても大丈夫

### **2. 品質を確認**

生成後、以下をチェック：
- ✅ ページの順序が正しいか
- ✅ キャラが統一されているか
- ✅ ストーリーが繋がっているか
- ⚠️ 修正が必要な部分があれば、個別のページ再生成機能で修正

### **3. データを定期的にバックアップ**

大作は紛失が痛いので、以下の対策を：
- PDF ダウンロードして保存
- 定期的にスクリーンショット撮影

## 📞 **トラブル解決**

### **Q: 100 ページを作成したら、途中で失敗した**

A: バックアップから復旧するか、個別ページを再生成してください。

### **Q: 生成中にページの統一感がなくなった**

A: キャラクター設定が途中で変わったためです。  
→ 「キャラクター再生成」で統一します

### **Q: 長編だと時間がかかりすぎる**

A: 以下の対策を：
- 夜間バッチ実行
- 短編を複数制作して組み合わせる
- 短編 → 続編 で段階的に拡張
